# Mantine code highlight

[![npm](https://img.shields.io/npm/dm/@mantine/code-highlight)](https://www.npmjs.com/package/@mantine/code-highlight)

Code highlight built with highlight.js and Mantine

[View documentation](https://mantine.dev/)

## Installation

```bash
# With yarn
yarn add @mantine/hooks @bds-react/core @mantine/code-highlight

# With npm
npm install @mantine/hooks @bds-react/core @mantine/code-highlight
```
